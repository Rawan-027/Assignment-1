# Bakery Sales Dataset - Exploratory Data Analysis (EDA)

## Dataset Source
This dataset was obtained from Kaggle:
https://www.kaggle.com/datasets/hosubjeong/bakery-sales

## Number of Rows and Columns
- Number of rows: 2422
- Number of columns: 27

## Description of Features (Columns)
- datetime: The date and time of each transaction.
- day of week: The day on which the transaction occurred.
- total: The total sales value for the transaction.
- place: The location where the purchase was made.
- product columns (e.g., americano, croissant, latte, etc.): Each column represents a specific product, where the value indicates the quantity sold in that transaction.

## Purpose of Using this Dataset
This dataset was selected to apply Exploratory Data Analysis (EDA) techniques.  
The goal is to analyze sales patterns, identify the most popular products, and understand customer behavior based on time and location.  
It also helps in identifying peak sales hours and days, which can support better business decision-making.
